## Методы строк (String Methods)
## Поиск (Search)
### includes() — проверяет наличие нужной подстроки в тексте.
### indexOf() — находит индекс первого вхождения символа.
## Извлечение (Extraction)
### slice() — вырезает часть строки по индексам.
### substring() — возвращает символы между двумя индексами.
## Преобразование (Transformation)
### trim() — удаляет лишние пробелы по краям.
### replace() — заменяет выбранный текст на новый.
### split() — разделяет строку на элементы массива.
### toLowerCase() — переводит все буквы в нижний регистр.
## Табдилдиҳӣ
### toString() — Ададро ба сатр табдил медиҳад.
### Number() — Қиматро ба адад табдил медиҳад.
## Яклухткунӣ
### floor() — Ададро ба поён яклухт мекунад.
### ceil() — Ададро ба боло яклухт мекунад.
### round() — Мувофиқи математика яклухт мекунад.
## Амалҳо
### max() — Адади калонтаринро меёбад.
### min() — Адади хурдтаринро меёбад.
### random() — Адади тасодуфӣ месозад.
### abs() — Модули ададро бармегардонад
